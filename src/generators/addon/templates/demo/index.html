插件调试方案

第一步：打开调试页面

选择编辑器菜单中的“开发者选项”->“开发者选项设置”

第二步：设置调试配制

填写调试设置配制必须配制参数

  • 包名：代码开发包中的package.json中的name字段的值
  • 域名：调试测试的地址，包开启调试后，默认地址为127.0.0.1:8020/index.js，一般无须修改
  • 启用：是否启用调试包，启用后，将会从加载该调试包

备注：设置后需要保存后刷新页面才能生效

第三步：开启并保证后刷新进行调试

启用并保存后，再刷新页面，会加载该调试文件，在network中查看是否加载，加载后调试成功，再验证功是生效

备注

如是页面是https,因跨域问题，则采用`tnpm run starts启动https服务,而非tnpm run start`启动http;
